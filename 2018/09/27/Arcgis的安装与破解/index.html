二.arcgis的安装与破解

arcgis的安装与破解分为3步：

1.安装 ArcGis License Manager
2.安装 ArcGis for Desktop
3.破解 ArcGis

下载好上面的文件后，我们就可以来安装了。首先我们用到的是arcgis安装包。把下载好的arcgis安装包解压，解压后目录如下：

双击上图标记的那个程序（ESRI.exe）后，可以得到如下界面。

1.安装 ArcGis License Manager

点击ArcGis License Manager那一行的set up。开始安装ArcGis License Manager。

安装完成后，界面如下。点击stop来停止服务。 ！！！这一点很重要，必须要停止服务后，再来进行接下来的安装步骤。

2.破解 ArcGis

我们找到刚才下载的破解文件，解压，得到如下两个文件。我们只需要用得到的这两个文件去替换原来的文件即可破解。

具体方法：选中这两个文件然后复制。再找到C:\Program Files\ArcGIS\License10.2\bin（ArcGis License Manager的安装目录）这个目录。右键粘贴，替换文件。

替换好文件后，我们又开启 ArcGis License Manager。就是点击lSAdmin.exe。（还是刚刚替换文件那个目录）
点击start开启第一大步关闭的服务，然后再点击Re-read Licenses。如图：

之后会弹出我们刚才关闭的那个界面。我们产品选择第一个，许可管理填入ocalhost。 点击确定。这样我们就完成了arcgis的安装与破解了。

3.安装 ArcGis for Desktop

接下来我们要安装的是ArcGis for Desktop。点击第一个set up就可以开始安装了。

ArcGis for Desktop的安装比较简单，直接下一步就OK。最后程序安装完成大约要花15分钟。安装完成后，会弹出如下界面，这时我们先不管它，直接x掉就好。然后我们开始arcgis的破解。

三.arcgis的汉化

解压汉化包，直接运行里面的程序，程序运行完成后即汉化完成。
